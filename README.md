
## Cài đặt

### Download
Download bộ source về máy và giải nén

### Env
Copy file .env.example thành file .env

Chỉnh cấu hình db trong file .env vừa tạo.

### Composer
    composer install

### npm
    npm install

### Migrate & seed
    php artisan migrate

    php artisan db:seed

### Run
    npm run dev

hoặc

    npm run build
